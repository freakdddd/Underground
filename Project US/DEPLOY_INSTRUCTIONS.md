1. **Buat akun GitHub**: https://github.com (jika belum punya)
2. **Buat repository baru**:

   - Klik "Create repository"
3. **Upload file**:
   - Di halaman repository, klik "uploading an existing file"
   - Drag semua file dari folder "Project US" ke browser
   - Klik "Commit changes"

   - Klik "Settings" di menu atas repository
   - Scroll ke bagian "Pages" di sidebar kiri

   - Pilih branch "main" dan folder "/ (root)"
   - Klik "Save"
5. **Website sudah online!**
   - URL akan seperti: `username.github.io/underground-soiree`
   - Butuh beberapa menit untuk pertama kali
